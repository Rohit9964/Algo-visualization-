# 🧮 Algorithm Visualization

An interactive web-based application that helps in understanding how different algorithms work through visual representation.

This project demonstrates the step-by-step execution of algorithms, making it easier to learn concepts like sorting and searching.

---

## 🚀 Features

- 🔢 Sorting Algorithms:
  - Bubble Sort  
  - Selection Sort  
  - Insertion Sort  
  - Merge Sort  
  - Quick Sort  

- 🔍 Searching Algorithms:
  - Linear Search  
  - Binary Search  

- 📊 Visual representation using bars and animations  
- ⏱️ Step-by-step execution of algorithms  
- 🔄 Shows comparisons, swaps, and iterations  
- 🎯 Simple and user-friendly interface  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## ⚙️ How It Works

- User selects an algorithm  
- Data is generated and displayed visually  
- Algorithm runs step-by-step  
- Each operation (comparison or swap) is shown using animation  

This helps in understanding how algorithms behave internally.

---

## ▶️ How to Run

1. Clone the repository

2. Open the project folder  

3. Run `index.html` in any browser  

---

## 🎓 Purpose

This project is useful for:
- Students learning Data Structures and Algorithms  
- Beginners preparing for coding interviews  
- Anyone who wants to understand algorithms visually
